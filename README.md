# TodoList-server
 

## ⚙️ Installation

```javascript
npm install
```

- create a database

Create an .env file, copy/paste the contents of the .env.example file and modify the DATABASE_URL with your configuration :

```javascript
DATABASE_URL="your configuration"
```

 -run the migration

```javascript
npx prisma migrate dev
```

## ✅ Run the project

```javascript
node index.js
```
